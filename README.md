Alertover Chome Extension
==========================

Alertover的chrome插件，用于接收推送。

![icon](/docs/images/alertover.jpg)

安装
-----

进入文件目录, 安装前端依赖

```bash
npm install
```

browserify编译js文件

```bash
make
```

添加到Chrome, 访问 chrome://extensions/ 
点击 'Load Unpacked Extension...' 然后选择alertover_extension文件夹
